# 提示词：回归验收清单

改完 `daily-plan-template.html` 后按此执行并勾选回复：

## Mock / 清空

- [ ] 清除全部后刷新全空  
- [ ] 仅「示例」产生演示；日+周完整；无产品自述  
- [ ] 清空本日：日空、周保留  
- [ ] 改星期：切到该日存档，不灌主题  

## 日闭环

- [ ] `→ 明日`：清单更新，日期不变  
- [ ] 开启新的一天：日期+1，MIT 承接  
- [ ] 次日有档：可合并 MIT  
- [ ] 跨周开新日且新周无档：主题日为空  

## 周 / focus

- [ ] 翻邻周不把本日重点写入邻周  
- [ ] 回本周主题/重点仍在  

## 日期星期

- [ ] 改日期 → 星期正确  
- [ ] 改星期 → 同周对应日  
- [ ] 选项顺序周一…周日  

## 审查 / 关机 / 导出

- [ ] 有跳过/待改进，非全绿冒充  
- [ ] 部分滚入 warn；全覆盖 ok  
- [ ] 关机双向  
- [ ] 预览与导出 MD/TXT 一致  

## UI

- [ ] 无紫渐变/光晕 AI 风  
- [ ] 主题按钮可持久  
- [ ] 下拉开后滚页面即关  
- [ ] 手机无整页左右滑
