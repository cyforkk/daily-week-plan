# 提示词：回归验收清单

改完 `daily-plan-template.html` 后按此执行并勾选回复。  
对照 skill：`make-plan-lessons`。

## Mock / 清空

- [ ] 清除全部后刷新全空  
- [ ] 仅「示例」产生演示；日+周完整；无产品自述  
- [ ] 清空**当前查看日**：该日空、日期不变、周保留  
- [ ] 查看非今天 → 清空 → 今天存档完好  
- [ ] 改星期：切到该日存档，不灌主题  

## 日闭环

- [ ] `→ 明日`：清单更新，日期不变  
- [ ] 开启新的一天：日期+1，MIT 承接；复盘/睡觉仪式清空  
- [ ] 次日有档：可合并 MIT  
- [ ] 跨周开新日且新周无档：主题日为空  

## 周 / focus

- [ ] 翻邻周不把本日重点写入邻周（保存后检查）  
- [ ] 日在 A、周面板在 B：**改输入框两侧不互写**  
- [ ] 回本周主题/重点仍在  
- [ ] 日∉周时预览无错误「今天主题」  

## 日期星期

- [ ] 改日期 → 星期正确  
- [ ] 改星期 → 同周对应日  
- [ ] 选项顺序周一…周日  

## 必做 / 审查 / 睡觉 / 导出

- [ ] 晨间未勾 → 待改进  
- [ ] 运动未选 / 不做无理由 → 待改进  
- [ ] 复盘未选 / 不复盘无理由 → 待改进  
- [ ] 睡觉仪式勾几项或设睡觉时间或「可以安心睡了」→ 可通过  
- [ ] 「可以安心睡了」不强制勾满所有仪式项  
- [ ] 有跳过/待改进，非全绿冒充  
- [ ] 部分滚入 warn；全覆盖 ok  
- [ ] 预览与导出 MD/TXT 一致（列表一条一项）  

## 条目列表

- [ ] +一项 / 删除 / Enter / 粘贴多行  
- [ ] 运动安排、边界、反思、感恩、明天均可  
- [ ] 旧换行文本能拆开  

## UI

- [ ] 无紫渐变/光晕 AI 风  
- [ ] 文案白话（无「深工」等主标签）  
- [ ] 主题按钮可持久  
- [ ] 下拉开后滚页面即关  
- [ ] 手机无整页左右滑
