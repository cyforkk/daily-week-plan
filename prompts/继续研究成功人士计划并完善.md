# 提示词：继续研究成功人士计划并完善模板

---

对照成功人士/高产方法（Ivy Lee、富兰克林、Newport 时间块与关机、Deep Work、Eat That Frog、Ferriss、GTD、艾森豪威尔等），审查 `daily-plan-template.html`：

1. 列出可复用**共性模块**（不要抄某人几点起床）。  
2. 对照现有字段与审查项，标缺口与过度设计。  
3. 只落地**轻量、可每天用、白话**的增强；遵守 `AGENTS.md` 与 mock/日闭环/条目列表硬约束。  
4. 更新 `docs/成功人士计划研究与审查.md`。  
5. 中文说明改了什么、为什么、刻意没做什么。  

## 优先检查（含历史坑）

- 日切诚实、mock 边界、审查可信、滚入语义  
- 是否又堆出「演示感」或难词  
- 关机/收工是否又变问卷；睡觉仪式应**任选不必堆满**  
- 新事项字段是否走条目列表 + `buildPlanView`  
- focus 表单门闩、清空当前查看日  

## UI

变更须去 AI 化、避免重复按钮、文案易懂。  

遵守 skills：`make-plan-product`、`make-plan-ui`、`make-plan-lessons`。
