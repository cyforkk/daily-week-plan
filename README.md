# 成功人士日/周计划 · 通用模板生成器

把成功人士计划方法里的**核心模块**收成可填写模板：

- **日计划**：意图、MIT（今日一件大事）、时间块、护栏、复盘、计划审查  
- **护栏**：非待办 · 延后 · 委派（艾森豪威尔轻量）  
- **关机四步**：开放循环 → 明日清单 → 扫日历 → 心理关机  
- **周计划 · 主题日**：主题 / 成功标准 / 周复盘  
- **→ 明日**（只写清单，不换日）、**开启新的一天**（真换日）
- **杂念 → 延后**

浏览器本地使用，无需姓名等个人档案。

## 快速开始

1. 打开 `daily-plan-template.html`
2. **日计划**：写 MIT → 时间块 → 护栏  
3. 日终：勾选完成 → **→ 明日** → 复盘 → **关机**  
4. **开启新的一天**：日期 +1，明日 MIT 成为新一天 MIT  
5. 顶栏切换 **周**；下方 **计划审查**（通过 / 待改进 / 跳过）  

数据保存在本机 `localStorage`（按日期、按周分档）。

## 功能一览

| 能力 | 说明 |
|---|---|
| 日计划 | 意图、MIT、晨间、时间块、杂念、健康、复盘 |
| MIT | 默认 3～6；#1=今日一件大事；完成勾选；→ 明日 |
| 时间块 | 深度/浅工作/会议/通信/健康/生活；骨架；缓冲审查 |
| 护栏 | 非待办 · 延后 · 委派 |
| 关机 | 四步清单 + 总确认 |
| 计划审查 | ~14 项实时检查 |
| 周计划 | 主题日 mon–sun；成功标准；周复盘；导出/打印 |
| 导出 | 日：打印/MD/TXT/复制；周：打印/MD |
