# 日序

**日序** — 浏览器里的日 / 周计划工具。

把可复用的计划结构（MIT、时间块、护栏、复盘关机、主题日）收成可填写模板：无安装、无后端、无账号，数据只在本机 `localStorage`。

> 不为「成功学」站台，只帮你把**今天和本周**排清楚。

- 主工具：`daily-plan-template.html`  
- 站点图标：`favicon.svg`  
- 仓库：`daily-week-plan`  

---

## 功能

| 模块 | 说明 |
|---|---|
| **日计划** | 意图、MIT、时间块、护栏、可选晨间/健康、复盘、关机 |
| **MIT** | 默认 3 条，最多 6；#1 = 今日一件大事 |
| **→ 明日** | 未完成写入明日清单（不改日期） |
| **开启新的一天** | 日期 +1，承接明日 MIT |
| **护栏** | 不做 · 延后 · 委派 · 杂念 |
| **关机** | 四步 + 总确认 |
| **计划审查** | 通过 / 待改进 / 跳过 |
| **周计划** | 主题日、成功标准、周复盘 |
| **导出** | 打印 / 复制 / MD / TXT |
| **主题** | 跟随系统 / 浅 / 深 |

---

## 本地使用

1. 浏览器打开 `daily-plan-template.html`  
2. 写 MIT → 时间块 → 护栏  
3. 日终：完成勾选 → **→ 明日** → 复盘 → 关机  
4. **开启新的一天**  
5. 顶栏切换 **周** 维护主题日  

「示例」填演示数据；「清空」只清本日。

---

## 部署 Netlify

见 `netlify.toml`：构建将工具与 `favicon.svg` 发布到 `dist/`。

1. Netlify 导入本仓库  
2. 自动读取构建配置  
3. 根路径即 **日序**  

无环境变量；数据仍在用户浏览器本地。
