# 成功人士日/周计划 · 通用模板生成器

把成功人士计划方法里的**核心模块**收成可填写模板：

- **日计划**：MIT、时间块、护栏、复盘、计划审查  
- **周计划 · 主题日**：周一至周日主题与焦点  
- **未完成滚入明日**：勾选完成 + 一键合并到明日清单  

浏览器本地使用，无需姓名等个人档案。

## 快速开始

1. 打开 `daily-plan-template.html`
2. **日计划**：写 MIT（默认可 3～6 件）→ 时间块 → 护栏  
3. 日终：勾选已完成的 MIT → 点 **未完成滚入明日** → 复盘/关机  
4. 顶栏切换 **周计划 · 主题日**：填本周主题，或「填入主题日预设」  
5. 可 **应用今日主题到日计划**、打印/导出  

数据保存在本机 `localStorage`。

## 功能一览

| 能力 | 说明 |
|---|---|
| 日计划 | 意图、MIT、晨间、时间块、杂念、健康、非待办、复盘 |
| MIT | 默认 3 槽，可加到 6；勾选完成；未完成滚入明日（去重≤6） |
| 时间块 | 深度/浅工作/会议/通信/健康/生活；典型日骨架 |
| 计划审查 | ~11 项实时检查（含滚入提醒） |
| 周计划 | 主题日 mon–sun；上周/下周；预设；同步本周重点；导出 MD/打印 |
| 导出 | 日：打印/MD/TXT/复制；周：打印/MD |
