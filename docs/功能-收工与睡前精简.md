# 功能：收工与睡前精简

## 背景

「关机仪式」原有无屏 / 阅读 / 冥想 + 杂事安顿 + 放下 + 睡前 + 总确认，勾选过多像问卷，与「易懂、轻量」冲突。

产品决策：**需要收工这个动作，不需要仪式大全**（方案 A）。

## 保留

| 控件 | 含义 |
|---|---|
| 杂事与临时想法已安顿 | 闭合开放循环 |
| 可以放下工作了 | 心理下班 |
| 睡觉时间 | 睡眠边界 |
| 今天工作结束 | 总确认；与上面两项双向 |

## 去掉（界面）

- 睡前少看屏幕 / 无屏
- 阅读
- 冥想 / 放松

`review.windDown` 仍可从旧存档读入，但不参与 UI、审查计数与总确认联动。

## 规则

- **双向**：两项全勾 → 自动勾「今天工作结束」；勾总确认 → 两项全勾；取消总确认 → 两项全清
- **审查**：勾 ≥1 项，或已设睡觉时间，或已总确认 → ok；否则建议「勾 1～2 项并设睡觉时间」
- **文案**：模块标题「晚间复盘与收工」；子区「收工与睡觉」

## 预览 / 导出

`shutBits` 仅含：杂事已安顿 · 可以放下工作 · 今天工作结束 · 睡觉 HH:mm
