# 功能：要事原子行动（导出说明）

## 1. 原子行动（行动步骤）

### 需求

「今天要做的事」通常 3 件左右。每件要事可拆成多条**可执行的具体一步**（原子行动），支持增加 / 删除 / 勾选完成。

### 数据

- `goals.mitSteps`：长度 6 的数组，与 `mits` / `mitDone` 下标对齐  
- 每项为 `{ text: string, done: boolean }[]`  
- 每件最多 12 步；旧存档无该字段时视为空数组  

### 交互

| 操作 | 行为 |
|---|---|
| + 一步 | 当前要事下追加空步骤并聚焦 |
| Enter | 在当前步下方插入新步 |
| × | 删除该步 |
| 勾选 | 完成该步（预览/导出同步） |

### 预览 / 导出

- 预览 / MD / TXT 缩进列出步骤  

## 2. 导出下载

MD / TXT / 周 MD 使用浏览器**默认下载**（`a.download` + UTF-8 BOM）。

**已移除**「选择导出文件夹 / 指定路径」功能（File System Access / IndexedDB 句柄等），避免 `file://` 与系统目录选择器问题。
