# 功能：睡觉仪式

日期：2026-07-18  
对象：`daily-plan-template.html`

## 背景

原「收工与睡觉」仅两项（杂事安顿 / 可以放下）+ 总确认，偏工作关机。  
产品改为 **睡觉仪式**：睡前可勾的身心动作列表，任选不必全勾。

## UI

模块标题：**晚间复盘与睡觉**  
子区：**睡觉仪式（任选几项，不必全勾）**

### 外链

- 心境日记：https://xinjing.cyforkk.top/  
- 睡觉仪式标题旁「打开日记网站」+ 下方「心境日记」说明（新标签页，`rel=noopener`）

## 规则

- 仪式项 **不** 与「可以安心睡了」强制双向（项多，全勾不合理）
- 审查（复盘 yes）：≥1 仪式项 **或** 已设睡觉时间 **或** 已勾「可以安心睡了」→ ok
- 旧 `windDown`（无屏/阅读/冥想）读入时迁入对应 `shutSteps`
- 旧仅 `inbox`/`phrase` 存档仍可读

## 预览 / 导出
