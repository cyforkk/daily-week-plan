## 1. 业务主流程（与代码一致）

```
日档 switchToDate  ←→  周档 switchToWeek
        │
        ▼
  MIT → 时间块 → 护栏 → 复盘/关机
        │
        ├─ → 明日（只写 tomorrowMit）
        └─ 开启新的一天（换日 + MIT 承接 / 有档可合并）
```

| 操作 | 实现要点 | 状态 |
|---|---|---|
| 改日期 / 改星期 | `switchToDate` 先存后载 | ✅ |
| 上·下周 / 本周 / 改周一 | `switchToWeek` | ✅ |
| 本周重点同步 | 仅 `dayBelongsToOpenWeek()` 时 `convergeWeekFocus` | ✅ |
| 滚入明日 | 不改日期；去重 ≤6 | ✅ |
| 开启新的一天 | 无档新建；有档加载+可选合并 MIT | ✅ |
| 跨周开新日无周档 | 空主题日，不克隆 | ✅ |
| 清空本日 | 清日留周 | ✅ |
| 示例 | 日+周 | ✅ |
| 关机双向 | 总确认 ⇔ 四步 | ✅ |

存储：`day-v2:{date}` · `week-v2:{monday}` · index / meta / legacy
