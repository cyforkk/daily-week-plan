# 成功人士日计划：研究归纳与模板审查

> 目的：从公开方法论中抽取**可移植结构**，对照 `daily-plan-template.html` 做缺口审查与迭代记录。  
> 原则：不抄某人完整作息表，只保留跨角色通用模块。

## 1. 主要参考来源（结构级）

| 来源 / 取向 | 核心做法 | 可移植模块 |
|---|---|---|
| **Ivy Lee Method** | 每晚写次日最多 6 件；排序；只做第一件直到完成 | MIT 上限 6；默认 3；明日清单；滚入 |
| **Benjamin Franklin** | 晨间「今日做什么有益」、分时段、晚间自省 | 今日意图；时间块；复盘问句 |
| **Cal Newport · Time Blocking** | 每晚排满次日时间块 | 时间块表；典型日骨架；缓冲审查 |
| **Deep Work** | 保护无干扰深工；浅工作隔离 | 类型：深度 / 浅工作 |
| **高产者通信习惯** | 邮件/消息 1～2 次集中处理 | 类型：通信 |
| **Miracle Morning (SAVERS)** | 静默/肯定/想象/运动/阅读/书写 | 晨间勾选 |
| **Eat That Frog** | 最难/最重要的事先做 | 今日一件大事 · MIT #1 |
| **Tim Ferriss** | Not-to-do；批处理 | 非待办；通信窗 |
| **GTD** | Capture；Weekly Review | 杂念卸载；周复盘字段 |
| **Eisenhower** | 做 / 安排 / 委派 / 删除 | 非待办 · 延后 · 委派（轻量，非完整四象限 UI） |
| **精力匹配**（实践派时间块） | 按能量档安排任务，而非只看优先级 | 能量 1–5 · 审查深工负荷 |
| **Newport Shutdown** | 闭合开放循环 + 计划明日 + 明确结束语 | 关机四步清单 |
| **主题日 / 周规划** | 每日主题；周成功标准；周日/周一规划 | 周计划视图 · 成功标准 · 周复盘 |
| **块间缓冲** | 防拖堂雪崩 | 审查 ≥15 分钟间隔 |

## 2. 共性提炼

1. **优先序**：有限清单 → 再排时间（Ivy Lee → Time Block）。  
2. **深度保护**：高峰给最重要结果；消息批处理。  
3. **分流**：不做 / 延后 / 委派，不全塞进今日。  
4. **捕获与闭环**：杂念先记；关机前处理开放循环。  
5. **精力匹配**：低能量缩短深工。  
6. **周层对齐**：主题日 + 成功标准 + 周回顾。  
7. **明确结束**：关机仪式降低工作 rumination。

| 缺口 | 本轮处理 |
|---|---|
| 艾森豪威尔只有非待办 | **+ 延后安排、委派/求助** 两栏 |
| 关机只是单勾选 | **+ 四步清单**（开放循环 / 明日 / 扫日历 / 心理关机） |
| MIT #1 不够突出 | 标签改为 **今日一件大事** |
| 周计划缺成功定义与回顾 | **+ 本周成功标准、周复盘** |
| 杂念与延后未连通 | **+ 杂念转入「延后」** |
| 缓冲仅靠文案 | **+ 审查：块间间隔 / 重叠** |
| 审查项偏少 | 扩展至约 **14 项** |
| 技巧 / 参考表 | + 延后委派、关机四步、缓冲、精力匹配 |

### 仍刻意不做

- 完整四象限拖拽看板  
- 番茄钟计时 / 日历 API  
- AI 自动生成计划  
- 跨周历史云端归档  

## 4. 推荐使用顺序

1. **周**（可选）：成功标准 · 主题日 · 本周重点  
2. **日对齐**：意图 · 今日一件大事 · MIT  
3. **锚定**：时间块骨架 · 通信窗 · 缓冲  
4. **护栏**：非待办 · 延后 · 委派 · 杂念  
5. **审查**：看未通过项  
6. **闭环**：复盘 · 滚入明日 · 关机四步
