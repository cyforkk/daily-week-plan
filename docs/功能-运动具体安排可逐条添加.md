# 功能：运动「具体安排」可逐条添加

> 已并入统一方案：见 [功能-条目列表一行一项.md](./功能-条目列表一行一项.md)。

## 需求

「运动与休息」模块里，具体安排不要整段大文本，而是**一条一条添加**：每条一个输入框，可增删。

## 怎么用

1. 选 **今天要运动**
2. 在「具体安排」下出现 **安排 1** 输入框
3. 点 **+ 一条安排** → 多出 **安排 2、3…**
4. 多于一条时，右侧 **×** 可删该条
5. 点 **+ 加入时间安排** → 每条有内容的安排各生成一个「运动休息」时间块
6. 预览 / 导出 MD / TXT / 审查 都会按列表展示

## 数据

| 字段 | 说明 |
|---|---|
| `health.movementPlans` | `string[]`，每条一条安排 |
| `health.movement` | 兼容旧数据：由有内容的 plans 用换行拼成 |

- 读档：优先 `movementPlans`；没有则从 `movement` 按行拆
- 示例：两条（晚饭后散步 / 力量训练）
- 清空：回到 `[""]` 一条空行

## 实现要点（`daily-plan-template.html`）

- UI：`#movementPlansList` + `#btnAddMovePlan`
- `ensureMovementPlans` / `renderMovementPlansList` / `syncMovementTextFromPlans`
- `buildPlanView` → `recovery.movementPlans`
- `auditPlan`：选了要运动则至少一条有内容
- 预览/MD/TXT：按列表输出
