# 清空：语义演进记录

## 历史

### 版本 1：沿用当前查看日

清空后日期保持 `state.meta.date`，只清空该日内容。

### 版本 2：强制回到今天（已废弃）

曾改为 `date = todayISO()`，以便「清空后总是当天」。

**问题**：若用户正在看明天/其它天再点清空：

- **今天**存档会被空状态覆盖  
- 屏幕上那一天未必清掉  
- 未落盘编辑可能丢失  

见 `docs/审查-代码与业务逻辑-2026-07-18.md` P0-2。

## 当前（方案 A，推荐）

**清空 = 清空当前正在查看的那一天**

```js
var date = state.meta.date || todayISO();
// next.meta.date = date;  // 日期不变
// 只写空 payload 到该 date
// week 完整保留
```

| 规则 | 行为 |
|---|---|
| 清空对象 | 当前查看日（不是「永远今天」） |
| 日期 | **不变** |
| 其它日期 | **不碰** |
| 周计划 | 完整保留 |
| 日∈打开周 | 清空后日侧 `weekFocus` 可从周 focus 回填展示 |
| 防抖保存 | 清空前 `cancelScheduledSave()` |

## 验收

1. 打开今天 → 填内容 → 清空 → 今天空、日期仍是今天、周仍在  
2. 打开明天 → 填内容 → 清空 → **明天**空、日期仍是明天、**今天**存档完好  
3. 确认框文案写明具体日期
