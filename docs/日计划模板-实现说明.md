# 日序 · 实现说明

## 功能概述

**日序**单文件 HTML 工具：把可复用的日计划结构收成模板，用户直接制定当日计划（目标 / 时间块 / 护栏 / 复盘），实时预览，并支持：

- 打印 / PDF  
- localStorage  
- 导出 Markdown / TXT、复制纯文本  
- **计划审查**（对照成功人士共性自动检查）

**定位**：制定计划，不是填写个人档案——不需要姓名、身份等字段。

- 交付文件：`daily-plan-template.html`
- 研究审查：`docs/成功人士计划研究与审查.md`
- 技术：原生 HTML + CSS + JS，无构建、无依赖

## 核心模块来源

| 参考取向 | 映射模块 |
|---|---|
| Ivy Lee 要事法 | MIT 默认 3、最多 6；明日多行 MIT |
| 富兰克林 | 今日意图；晚间「完成了什么」 |
| Time Blocking / Deep Work | 时间块；深度/浅工作；典型日骨架 |
| 高产者通信习惯 | 时间块类型「通信」 |
| Miracle Morning | 晨间 SAVERS 勾选 |
| Ferriss 非待办 | 非待办清单 |
| GTD 捕捉 | 杂念卸载 |
| 精力管理 + 关机仪式 | 能量/健康；关机勾选 |
| 目标对齐 | 年/季 → 本周 → 今日 |

## 数据结构（state）

```js
{
  meta: { date, weekday, energy, theme, intention },
  goals: {
    yearOrQuarter, weekFocus, mitCount: 3,
    mits: ["","","","","",""],
    mitDone: [false,false,false,false,false,false]
  },
  morning: { items: {...}, note },
  blocks: [{ start, end, type, content, outcome }],
  health: { sleepHours, movement, meals },
  notToDo: "",
  capture: "",
  week: {
    weekStart, // 周一 YYYY-MM-DD
    focus,
    days: {
      mon: { theme, focus, note },
      tue: {...}, wed: {...}, thu: {...}, fri: {...}, sat: {...}, sun: {...}
    }
  },
  review: { done, gap, learned, gratitude, tomorrowMit, shutdown }
}
```

- `type`：`deep` | `shallow` | `meeting` | `comm` | `health` | `life`
- `mitCount`：3–6；`mitDone` 与 mits 对齐
- **滚入明日**：未完成 MIT → `review.tomorrowMit`（去重、截断 6）
- **周计划**：顶栏切换「周计划 · 主题日」；可打印/导出周 MD
- 本地存储键：`daily-plan-template-v1`

## 计划审查

`auditPlan(state)` → 约 10 项：MIT、意图/主题、深度块、先吃青蛙、通信窗、非待办、周/季对齐、能量负荷、明日 MIT、关机。

- 在 `renderPreview` 末尾调用 `renderAudit()`
- 仅提醒，不阻断导出

## 生成 / 导出规则

1. **预览**：整块全空不渲染该节  
2. **打印**：隐藏表单、导航、审查、参考、技巧  
3. **MD/TXT**：UTF-8 BOM  
4. **复制**：clipboard API + execCommand 兜底  

## 界面分区

1. 四步制定顺序（对齐 → 锚定 → 护栏 → 闭环）  
2. 左侧表单 + 右侧预览  
3. 计划审查  
4. 参考对照表 + 技巧卡片  

## 遇到的问题与处理

| 问题 | 处理 |
|---|---|
| file:// 剪贴板受限 | 复制失败提示导出 TXT |
| Windows 中文 TXT 乱码 | UTF-8 BOM |
| 打印混入表单 | `no-print` + `@media print` |
| 日期星期不同步 | 本地时区算星期；星期为下拉 |
| 不要姓名身份 | 移除 name/role |
| MIT 3 还是 6 | 默认 3 槽，可加到 6 |
| 研究后模块缺口 | 意图、通信、骨架、杂念、关机、审查 |
| 滚入明日 + 周计划 | MIT 完成勾选与一键滚入；周视图主题日预设与同步 |
| 日期与星期双向绑定 | 改日期→星期；改星期→同一周对应日期；二者不用 data-path，专用监听强制写回控件 |
| 下拉框现代化 | 自定义 cselect：选项纯文字 + 触发器箭头图标 |
| 手机填示例横向溢出 | 单栏默认 + minmax(0,1fr)；预览表 table-layout:fixed；表格区内滑；取消手机 sticky |
| 研究完善 2 | 延后/委派、关机四步、周成功标准与复盘、杂念转延后、缓冲审查、今日一件大事 |
| 审查闭环修复 | 开启新的一天；按日/周分档；审查三态；关机双向；周重点自动同步 |
| UI 全面现代化 | 设计 token、毛玻璃顶栏、圆角按钮/分区卡片、现代确认弹窗替代 confirm、Toast 升级 |
| 夜间模式 | 顶栏按钮循环：跟随系统 → 浅色 → 深色；`localStorage` 键 `daily-plan-theme-v1`；系统模式时监听系统变化 |
