# UX 精简说明

从「每天打开写计划」出发，去掉教学感、重复入口与低频噪音。

## 去掉 / 合并了什么

| 原设计 | 处理 | 原因 |
|---|---|---|
| 大段 Hero + 四步流程卡 | 删除 | 与表单重复，占首屏 |
| 顶栏 6 个锚点 | 只留「审查 / 方法参考」 | 日/周已有 Tab |
| 「未完成→明日」3 处 | 表单 1 处 + 手机底栏 | 防重复 |
| 日表单「套用周主题」 | 只保留周视图 | 入口单一 |
| 各区长教学 hint（Ferriss/Newport…） | 删掉 | 方法放底部参考 |
| 晨间 + 健康常显 | 默认折叠 | 低频 |
| 杂念独立大区块 | 并入「护栏」 | 同一决策区 |
| 桌面底部粘性条 | 仅手机显示 | 桌面顶栏已够 |
| 方法参考 / 小技巧默认展开 | 默认收起 | 写计划时不挡视线 |
| 按钮文案过长 | 缩短（示例/清空/→明日） | 扫读更快 |

## 保留的核心路径

```
概览 → MIT → 时间块 → 护栏 → 复盘关机 → 开启新的一天
```

可选：晨间/健康、周主题、审查、方法参考。

## 输入原则

- Label 短；placeholder 只补「可选 / 每行一项」
- 复盘 2×2 网格，减少纵向滚动
- 关机勾选一行一句，去掉冗长括号说明
